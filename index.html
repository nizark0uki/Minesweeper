How to Play Minesweeper

Minesweeper is a classic puzzle game where the objective is to clear a board containing hidden mines without detonating any of them, with help from clues about the number of neighboring mines in each field.

  • The game board consists of a grid of squares, some of which contain mines.
  • Left click on a square to reveal what is underneath it.
  • If you reveal a mine, you lose the game.
  • If you reveal an empty square, a number may appear in the square. This number indicates how many of the surrounding squares contain mines.
  • Use these numbers to deduce which nearby squares contain mines and which are safe to click.
  • ctrl and left click (or long press on mobile) to place a flag on squares you believe contain mines.
  • Click the smiley face to Restart the game
  • The game is won when all mines are flagged correctly.
